# 游戏对象介绍

## 基础建筑

| 名称     | 功能                             | 说明                           |
| -------- | -------------------------------- | ------------------------------ |
| 基地核心 | 可以建造机器人，存储资源等       | 建筑在基地核心的范围内才能工作 |
| 煤矿     | 可以用作发电                     |                                |
| 采集器   | 自动采集煤矿中的能量             |                                |
| 储存器   | 储存资源                         |                                |
| 传送器   | 可以传送资源                     |                                |
| 钓鱼机   | 可以抛出鱼线，钓到鱼后会自动收回 |                                |

## 电力建筑

| 名称       | 功能                           | 说明 |
| ---------- | ------------------------------ | ---- |
| 能源发电机 | 消耗能量矿产生电力             |      |
| 小电池     | 存储电力                       |      |
| 大电池     | 存储电力                       |      |
| 电力节点   | 连接发电机和用电建筑，传递电力 |      |

## 防御塔建筑

| 名称     | 功能             | 说明 |
| -------- | ---------------- | ---- |
| 普通炮塔 | 发射炮弹攻击敌人 |      |
